# Responsive Landing Page

## Description

A simple and responsive landing page created using HTML and CSS. It includes a Hero section, About section, Services section, and Contact section.

## Features

* Responsive design
* Hero section
* About section
* Services section
* Contact section
* Smooth scrolling
* Hover effects

## Technologies Used

* HTML5
* CSS3

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in your web browser.
